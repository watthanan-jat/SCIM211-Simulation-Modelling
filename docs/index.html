Course Description

Simulation As A Way To Study Real Systems

Simulation modelling is used when real systems are too complex, expensive, risky, or time-consuming to study directly. This course introduces how to represent a system, describe uncertainty, generate random behaviour, run computational experiments, and interpret simulation output.

The course combines mathematical ideas with applied Python work. Weekly theory sessions introduce the modelling and statistical concepts, while the lab sessions develop a continuing coffee-shop simulation that grows from a simple queue into a tool for comparison, validation, and decision-making.

Semester Project

Coffee Shop Simulation Project

The project models customer arrivals, drink orders, service times, queue behaviour, profit, customer loyalty, and uncertainty. Each lab adds one new modelling idea to the same familiar system.

  • Build small Python simulations step by step.
  • Connect probability concepts to practical decisions.
  • Use plots, tables, and confidence intervals to explain results.
  • Write clear assumptions, limitations, and recommendations.

Mini Project

Coffee Shop Simulation Decision Challenge

Build a reusable coffee-shop simulation model, compare a baseline policy with an improvement policy, and respond to an unseen operational stress test near the end of the course.

View Project Outline

Build

Model arrivals, drink choices, service times, baristas, queues, outputs, and optional loyalty behaviour.

Compare

Evaluate a baseline system against a policy such as adding a barista, simplifying the menu, or running a promotion.

Defend

Use replications, confidence intervals, validation checks, and limitations to support a practical recommendation.

Course Learning Outcomes

What This Course Builds

CLO1

Explain the fundamental concepts and importance of simulation modelling in solving complex problems.

CLO2

Generate random numbers and random variates from specified probability distributions using standard computational techniques.

CLO3

Implement Monte Carlo simulation methods and discrete-event models to solve problems in various application domains.

CLO4

Analyse simulation inputs and outputs using appropriate statistical techniques to create accurate model representations and draw valid conclusions.

CLO5

Construct Markov chain models to represent stochastic processes in real-world systems.

CLO6

Evaluate simulation models through systematic validation approaches and appropriate tools for different problem domains.
