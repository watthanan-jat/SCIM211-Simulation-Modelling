# Schedule

SCIM211 runs for 15 weeks. Each week includes a 2-hour theory session. Every even week also includes a 2-hour lab connected to the semester-long Coffee Shop Simulation Project.

| Week | Theory Topic | Lab |
|---:|---|---|
| 1 | Introduction | - |
| 2 | System concepts | Lab 1 |
| 3 | Probability review | - |
| 4 | Random number generation | Lab 2 |
| 5 | Simulating random variables I | - |
| 6 | Simulating random variables II | Lab 3 |
| 7 | Monte Carlo I | - |
| 8 | Monte Carlo II | Lab 4 |
| 9 | Midterm/review | - |
| 10 | Discrete-event simulation I | Lab 5 |
| 11 | Discrete-event simulation II | - |
| 12 | Markov chains I | Lab 6 |
| 13 | Markov chains II | - |
| 14 | Input/output analysis | Lab 7 |
| 15 | Variance reduction, validation, and project wrap-up | - |

## Lab Theme

All labs use the Coffee Shop Simulation Project. The project is developed step by step so that students can connect theory topics to one consistent modelling problem.

## Notes

The schedule may be adjusted if the class needs additional time for review, project support, or assessment preparation. Any changes should preserve the main sequence of topics and the Coffee Shop Simulation Project structure.
