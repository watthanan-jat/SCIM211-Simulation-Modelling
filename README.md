# SCIM211 Simulation Modelling

SCIM211 Simulation Modelling is a second-year course for Industrial Mathematics and Data Science students. The course introduces simulation as a practical method for studying systems that involve uncertainty, randomness, queues, decisions, and performance measurement.

Students will learn the mathematical ideas behind simulation and apply them through a semester-long Coffee Shop Simulation Project. The project develops gradually across the labs, giving students repeated practice in modelling, coding, analysing, and communicating simulation results.

## Course Format

- Duration: 15 weeks
- Theory: 2 hours every week
- Lab: 2 hours every even week
- Lab project: Coffee Shop Simulation Project

## Repository Guide

- `syllabus.md`: course aims, learning outcomes, and expectations
- `schedule.md`: weekly theory and lab schedule
- `assessment.md`: assessment structure and submission expectations
- `ai_policy.md`: guidance on acceptable AI use in the course
- `lectures/`: lecture materials
- `exercises/`: practice questions and short activities
- `labs/`: Coffee Shop Simulation Project lab sheets
- `data/`: sample datasets and project data
- `src/`: reusable code and simulation examples
- `tests/`: tests for project or example code

## Student Expectations

Students are expected to attend theory sessions, complete lab work on schedule, maintain their project files carefully, and explain their own modelling decisions. The emphasis is not only on getting code to run, but also on understanding what the simulation represents and how results should be interpreted.
