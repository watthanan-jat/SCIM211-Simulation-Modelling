Welcome

SCIM211 is built around one continuing Coffee Shop Simulation Project. We begin with a simple question, such as how long customers wait, then gradually add randomness, queues, decisions, data analysis, and recommendations.

Semester Project

Coffee Shop Simulation Project

The project models customer arrivals, drink orders, service times, queue behaviour, profit, customer loyalty, and uncertainty. Each lab adds one new modelling idea to the same familiar system.

  • Build small Python simulations step by step.
  • Connect probability concepts to practical decisions.
  • Use plots, tables, and confidence intervals to explain results.
  • Write clear assumptions, limitations, and recommendations.

Learning Objectives

What This Course Builds

Model Systems

Translate a real situation into entities, resources, events, activities, state variables, inputs, and outputs.

Use Randomness

Generate random numbers, simulate random variables, and explain why reproducibility matters.

Run Experiments

Build Monte Carlo and discrete-event simulations for queueing, profit, loyalty, and validation questions.

Explain Results

Use tables, plots, confidence intervals, assumptions, and limitations to support clear recommendations.

Lab Pathway

What You Will Build

Lab 1

From Real System to Simulation Model

Describe the coffee shop system and calculate a first simple queue.

 Lab 2

Random Numbers

Generate arrivals and inspect random-number quality.

 Lab 3

Random Variables

Simulate drink orders and drink-dependent service times.

 Lab 4

Monte Carlo

Estimate profit, risk, and a matcha promotion decision.

 Lab 5

Discrete-Event Simulation

Compare calm, normal, rush, and overloaded queue scenarios.

 Lab 6

Markov Chains

Model how customers move between loyalty states.

 Lab 7

Validation

Use data, replications, confidence intervals, and a final recommendation.
